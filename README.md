# Coffee Sales Dashboard (Excel Project)

## 📊 Overview
This project analyzes coffee sales data using Excel.

## 🔧 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Visualizations

## 📈 Features
- Sales trends analysis
- Region-wise performance
- Product category insights

## 📂 File
- coffee_sales_dashboard.xlsx

## 🚀 Insights
- Identified top-selling products
- Seasonal trends in sales
## 🎯 Business Problem
The goal of this project is to analyze coffee sales data to identify trends, top-performing products, and regional performance.

## 📊 Key Insights
- Top-performing products contribute to majority of revenue
- Certain regions show higher sales growth
- Seasonal patterns impact sales significantly

## 💼 Business Impact
- Helps in inventory planning
- Improves targeted marketing strategies
- Supports data-driven decision making
